## RULES FOR AppColors

- Use plain `Color`, never a custom wrapper.
- Include a `factory AppColors.light()` and a `factory AppColors.dark()`.
- Dark mode rules:
  - Background tokens: darken significantly (e.g. bgColor ‚Üí #0E0E0E, cards ‚Üí #1A1A1A, overlays ‚Üí #2A2A2A)
  - Text tokens: invert (headings near-white, body/subtext scaled down in brightness)
  - Semantic tokens (error, success, inputBorderFocus): keep identical or subtly adjust for contrast
  - Brand gradients and color scales (blue, purple, etc.): keep unchanged ‚Äî they are brand identity tokens
- Every color token from the input must appear in both `light()` and `dark()` factories.
- Include all gradients as `LinearGradient` fields. Preserve `GradientRotation`, `stops`, `begin`, `end` exactly as provided.
- Include all color scale palettes (e.g. blue50‚Äìblue900, purple50‚Äìpurple900) as individual `Color` fields.

## RULES FOR AppSpacing

- All spacing and radius values are theme-agnostic. Use a single `factory AppSpacing.defaultScale()`.
- Add `BorderRadius` convenience getters for every radius token.
- Name spacing fields `s1`‚Äì`sN` (not `spacing1`). Keep a `base` field for the spacing unit.

## RULES FOR AppTextStyles

- Use a single `factory AppTextStyles.defaultStyles()` ‚Äî text styles don't change between themes.
- Base all styles on a shared `static const _base = TextStyle(leadingDistribution: TextLeadingDistribution.even)`.
- Preserve fontFamily, fontSize, fontWeight, fontStyle, height, letterSpacing, decoration exactly as provided.

## RULES FOR ALL ThemeExtension classes

- Annotate every class with `@immutable`.
- Implement `copyWith(...)` with every field nullable.
- Implement `lerp(...)` using Flutter's built-ins:
  - Colors ‚Üí `Color.lerp`
  - Gradients ‚Üí `LinearGradient.lerp`
  - TextStyles ‚Üí `TextStyle.lerp`
  - Doubles ‚Üí `lerpDouble`
- No custom lerp helpers or extension methods.

## RULES FOR AppTheme

- Extract shared extensions into a private `static List<ThemeExtension<dynamic>> get _sharedExtensions` getter containing `AppSpacing` and `AppTextStyles`.
- `light` uses `ThemeData.light(useMaterial3: true).copyWith(...)`
- `dark` uses `ThemeData.dark(useMaterial3: true).copyWith(...)`
- Both inject `AppColors.light()` / `AppColors.dark()` plus `..._sharedExtensions`.
